# 概述

此為 RWD 網頁可以在不同裝置上瀏覽操作，你可以在網站上瀏覽房間資訊，並預訂房間，也可成為一個房東將自己閒置的房間出租。

# 功能

#### 登入登出 (jwt token)

#### 可修改會員資訊

#### 可瀏覽出租房間列表 - 使用 lazy loading， 達成資料延遲載入

#### 房源管理 - 新增及修改

#### 訂單管理 - 通知租客確認訂單

#### 查看訂單及行程

#### 評價住宿
